#  College Event Management System

##  Overview
The College Event Management System is a database-driven application designed to simplify the organization of college events. It automates tasks like event creation, participant registration, and result management, reducing manual effort and errors.

##  Features
- Admin can create and manage events  
- Student event registration  
- Coordinator assignment  
- Event updates and result publishing  

##  Technologies Used
- DBMS  
- SQL  
- Relational Database Design  

##  How to Run
1. Set up the database  
2. Run the application
